## 🔍 如何验证名称

### 方法1：在代码中检查
```python
import ccxt

# 列出所有支持的交易所
print(ccxt.exchanges)

# 检查特定交易所
if 'binance' in ccxt.exchanges:
    print("✓ 支持")
```

### 方法2：在线查询
访问 [CCXT 文档](https://github.com/ccxt/ccxt/wiki/Manual#exchanges)

## 🎯 常见错误

### ❌ 错误示例
```json
{
  "exchange_type": "LBANK"  // 错误！大写
}
```

```json
{
  "exchange_type": "Binance"  // 错误！首字母大写
}
```

```json
{
  "exchange_type": "Gate.io"  // 错误！有点号
}
```

### ✅ 正确示例
```json
{
  "exchange_type": "lbank"  // 正确！全小写
}
```

```json
{
  "exchange_type": "binance"  // 正确！全小写
}
```

```json
{
  "exchange_type": "gateio"  // 正确！没有点号
}
```

### 添加账户时
```

2. 在 "交易所类型" 下拉框中选择
3. 或手动输入（必须小写）

正确输入：
✓ binance
✓ okx
✓ bitget
✓ lbank

错误输入：
✗ BINANCE
✗ OKX
✗ Bitget
✗ LBANK
```

## 🔧 自动修复

程序已添加自动转换功能：
```python
# 程序会自动将名称转换为小写
exchange_type = account.exchange_type.lower()
```

**但建议直接使用正确的小写名称！**

## ❓ 常见问题

### Q1: 为什么必须小写？
**A:** ccxt 库要求交易所名称必须是小写，这是标准规范。

### Q2: 输入大写会怎样？
**A:** 程序现在会自动转换为小写，但建议直接使用小写。

### Q3: 如何查看所有支持的交易所？
**A:** 运行：
```python
import ccxt
print(ccxt.exchanges)
```

### Q4: 我的交易所不在列表中？
**A:** 检查 [CCXT 文档](https://github.com/ccxt/ccxt) 确认是否支持。

## 🔄 更新配置

如果你已经添加了错误的交易所名称：

1. 打开 **多交易所管理界面**
2. 点击 **"✏️ 编辑账户"**
3. 选择要修改的账户
4. 修改 **"交易所类型"** 为小写
5. 点击 **"💾 更新账户"**

或者直接编辑 `exchanges_config.json` 文件，将 `exchange_type` 改为小写。
