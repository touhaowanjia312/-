## 🚀 快速开始

### 第一步：安装 GUI 依赖

```bash
pip install -r requirements.txt
```

这将安装：
- `customtkinter` - 现代化 UI 库
- `pillow` - 图像支持
- `matplotlib` - 图表库（预留）

### 第二步：启动 GUI

**Windows 用户：**
```bash
# 方法 1: 双击文件
start_gui.bat

# 方法 2: 命令行
python gui_main.py
```

**Linux/Mac 用户：**
```bash
# 添加执行权限（首次）
chmod +x start_gui.sh

# 启动
./start_gui.sh

# 或直接运行
python3 gui_main.py
```

### 第三步：探索功能

1. **无需配置即可测试**
   - 点击 "🧪 信号测试" 标签
   - 测试信号解析功能

2. **配置后启动机器人**
   - 填写 Telegram 配置
   - 点击 "▶ 启动机器人"
   - 在终端输入验证码（首次）

3. **观察运行状态**
   - 查看 "📝 实时日志"
   - 监控统计信息
   - 刷新账户余额
