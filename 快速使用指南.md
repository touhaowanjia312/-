## 🎨 界面功能说明

### 左侧控制面板

#### 1. 状态指示器
- **● 未运行** (灰色) - 机器人未启动
- **● 运行中** (绿色) - 机器人正在运行
- **● 已停止** (红色) - 机器人已停止

#### 2. 控制按钮
- **▶ 启动机器人** - 启动 Telegram 监听
  - ⚠️ 需要先配置 .env 文件
  - ⚠️ 首次运行需要输入验证码（在终端窗口）
- **■ 停止机器人** - 停止运行

#### 3. 统计信息
- **识别信号** - 成功识别的信号数量
- **执行交易** - 实际执行的交易数量
- **总盈亏** - 累计盈亏（需要交易所API）
- **胜率** - 盈利交易比例

#### 4. 账户余额
- 显示 USDT 余额
- 点击 "🔄 刷新余额" 更新（需要交易所API）

#### 5. 快速配置
- **启用交易** ☐ - 关闭时仅监听，不执行交易
- **使用测试网** ☑ - 建议保持开启
- **风险百分比** - 拖动滑块调整（0.1% - 5.0%）

#### 6. 主题切换
- **深色模式** ☑ - 护眼模式（推荐）
- **浅色模式** ☐ - 明亮模式

### 右侧标签页

#### 📝 实时日志
- 显示程序运行的所有日志
- 自动滚动到最新
- 点击下方 "清除日志" 可清空

#### 🧪 信号测试 ⭐推荐先试这个
- **输入框** - 输入或粘贴交易信号
- **测试按钮** - 点击测试解析
- **结果显示** - 显示详细的解析结果

#### 📈 交易历史
- 表格形式显示所有交易记录
- 包含时间、交易对、类型、价格、数量、状态

#### ⚙️ 详细设置
- **Telegram 配置** - API ID、Hash、手机号、群组ID
- **交易所配置** - 交易所名称、API Key、Secret
- **保存按钮** - 保存配置到 .env 文件

## 💡 使用技巧

### 技巧 1: 信号测试是最安全的开始
在配置任何 API 之前，先在 "🧪 信号测试" 中测试你的群组信号格式。

### 技巧 2: 从监听模式开始
第一次配置好 Telegram 后，保持 "启用交易" 关闭，先观察几天信号识别情况。

### 技巧 3: 使用测试网
配置交易所 API 时，先使用测试网（Testnet），避免真实资金损失。

### 技巧 4: 小额开始
